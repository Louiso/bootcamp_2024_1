# Generador de Mensajes

Generador de mensajes multicanal (Email, SMS, WhatsApp) con interfaz web.

##  Pasos para Ejecutar

1. **Clonar el Repositorio**

```bash
git clone https://github.com/Louiso/bootcamp_2024_1.git

```

```bash
npm install
# o si usas yarn
yarn install
```
